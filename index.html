Hospedagem de Sites

Passos pra hospedar

  • Criar repositório no remoto (Github);
  • Criar repositório local: git init;
  • Adicionar arquivo sem zona de preparação: git add;
  • Criar ponto na história: git commit -m "";
  • Renomear a branch para main: git branch -M;
  • Unificar repostiórios: git remote add origin [endereço-repositórios github];
  • Enviar o código: git push -u origin main
